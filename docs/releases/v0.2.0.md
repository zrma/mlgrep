# mlgrep 0.2.0

mlgrep 0.2.0 is the bounded-memory streaming release.

Status: published on 2026-07-17 at
<https://github.com/zrma/mlgrep/releases/tag/v0.2.0>.

## Highlights

- Files are processed incrementally through Mallang 1.1.0
  `std/fs.forEachLine[C,S]`; the default runtime path no longer retains or
  splits the full input.
- Exact UTF-8 substring matching, 1-based `line:content` output, `--count`, and
  exit classes `0`, `1`, and `2` remain compatible with v0.1.0.
- Empty files, terminal LF, a final line without LF, Unicode, invalid UTF-8, and
  missing-file behavior have native CLI regressions.
- Deterministic 1, 10, and 100 MiB inputs verify count output and bounded peak
  RSS on macOS arm64 and Linux x86_64.
- Deterministic native archives, combined SHA-256 checksums, and the atomic
  installer retain the v0.1.0 distribution contract.

## Install

```sh
curl -fsSLO https://github.com/zrma/mlgrep/releases/download/v0.2.0/install.sh
sh install.sh --version 0.2.0
```

The default destination is `$HOME/.local/bin/mlgrep`.

## Verify

```sh
mlgrep --version
mlgrep ERROR tests/fixtures/sample.log
scripts/check-published-release.sh --version 0.2.0
```

The signed tag points at the synchronized streaming implementation, Mallang
1.1.0 dependency, memory evidence, version contract, release tooling, supported
platform archives, checksums, installer, and release notes.
