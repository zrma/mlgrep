# mlgrep 0.3.0

mlgrep 0.3.0 completes the grep-shaped Mallang reference workload with
deterministic multiple-file streaming.

Status: published on 2026-07-17 at
<https://github.com/zrma/mlgrep/releases/tag/v0.3.0>.

## Highlights

- One or more explicit file paths are processed sequentially in command-line
  order without retaining whole inputs.
- Single-file output remains compatible with v0.2.0. Multiple-file matches use
  `path:line:content`, while count mode emits one ordered `path:count` record per
  operand, including zero-count and duplicate paths.
- Exit `0` means any file matched, exit `1` means every file was processed with
  no match, and exit `2` stops at the first usage, read, or output error.
- Native CLI regressions cover ordering, duplicate operands, aggregate no-match,
  zero counts and output retained before a later read error.
- A deterministic sequential 200 MiB aggregate workload joins the 1, 10 and
  100 MiB bounded-memory gate on macOS arm64 and Linux x86_64.
- The product boundary explicitly positions mlgrep as Mallang's external
  reference implementation, not a claimed replacement for `grep` or `ripgrep`.

## Install

```sh
curl -fsSLO https://github.com/zrma/mlgrep/releases/download/v0.3.0/install.sh
sh install.sh --version 0.3.0
```

The default destination is `$HOME/.local/bin/mlgrep`.
