# mlgrep 0.1.0

This is the first native binary release of mlgrep, an external Mallang v1 dogfood project.

Status: published on 2026-07-17 at
<https://github.com/zrma/mlgrep/releases/tag/v0.1.0>.

## Highlights

- Exact UTF-8 substring search with 1-based `line:content` output.
- `--count` mode without rendered per-line output allocation.
- Match, no-match and usage/I/O exit classes `0`, `1` and `2`.
- Deterministic native archives for macOS arm64 and Linux x86_64.
- SHA-256 verification and atomic installation into a user-selected binary directory.

## Install

```sh
curl -fsSLO https://github.com/zrma/mlgrep/releases/download/v0.1.0/install.sh
sh install.sh --version 0.1.0
```

The default destination is `$HOME/.local/bin/mlgrep`.

## Verify

```sh
mlgrep --version
mlgrep ERROR tests/fixtures/sample.log
```

The signed tag points at the version contract, release tooling, supported-platform archives,
checksums, installer and release notes verified by the release workflow.
