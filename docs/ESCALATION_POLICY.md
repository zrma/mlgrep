# Escalation Policy

The agent proceeds autonomously for local, reversible implementation, tests, documentation and
`jj` change preparation when the product contract is already explicit.

Escalate only for:

- a source-visible Mallang compatibility or language-design change;
- regex/case-folding/streaming semantics that alter the declared product contract;
- credentials, private context, cost or destructive operations;
- remote repository creation, push, tag, release, visibility change or published history rewrite;
- a blocker that cannot be reproduced or resolved with repository-owned evidence.

Ask for the smallest concrete decision and preserve all unaffected local progress.
